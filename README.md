# react-avatar-firebase

> Easily Avatar on Firebase Storage

[![NPM](https://img.shields.io/npm/v/react-avatar-firebase.svg)](https://www.npmjs.com/package/react-avatar-firebase) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save react-avatar-firebase
```

## Usage

```jsx
import React, { Component } from 'react'

import { useMyHook } from 'react-avatar-firebase'

const Example = () => {
  const example = useMyHook()
  return (
    <div>{example}</div>
  )
}
```
